================================================================================
                            TROUBLESHOOTING
================================================================================

ERROR: 401 Unauthorized
→ Baca: SOLUSI_401_UNAUTHORIZED.md
→ Penyebab: Token di tempat salah (query params/body, bukan header)
→ Solusi: Tambah Authorization header dengan format "Bearer <token>"

ERROR: Database connection failed
→ Penyebab: PostgreSQL tidak running
→ Solusi: Buka Services, cari PostgreSQL, klik Start

ERROR: Cannot find module
→ Penyebab: Dependencies belum diinstall
→ Solusi: npm install

ERROR: Token tidak valid
→ Penyebab: Token sudah expired atau JWT_SECRET tidak cocok
→ Solusi: Login ulang untuk dapatkan token baru

ERROR: Akses ditolak (403)
→ Penyebab: Role user tidak sesuai dengan endpoint
→ Solusi: Gunakan user dengan role yang tepat

ERROR: Data tidak lengkap (400)
→ Penyebab: Field required tidak diisi
→ Solusi: Cek contoh request di QUICK_REFERENCE.md
