  HOW TO RUN
================================================================================

   npm install

2. Update IP address in services/api.js (line 7):
   const API_BASE = 'http://YOUR_IP:3000/api';

   cd apps/backend
   node app.js

4. Start mobile app:
   cd apps/mobile/module2
   npx expo start

5. Login with existing credentials
